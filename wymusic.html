网易云音乐下载器 使用说明

第一步 下载虚拟Linux系统

下载并安装Termux或ZeorTermux（Termux请到Google Play下载）

ZeroTermux（推荐）

第二步 安装基本环境

打开刚刚下载的应用，直接运行以下命令

cd ~ && sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list && apt update && apt upgrade &&

pkg install python git && pip install requests bs4 -i https://pypi.tuna.tsinghua.edu.cn/simple && git clone https://github.com/xiaobai00000/musicDownload.git

运行后，需要选择的时候，直接回车

第三步 正式使用

运行以下命令即可使用

cd ~ && cd musicDownload && python download.py

在您第一次运行时，需要设置下载目录（不会的请直接回车跳过）

在输入分享内容时，请到网易云音乐登记分享音乐并选择复制链接

然后直接将剪贴板的内容粘贴进去，一个字都不能删除

下载完成后会有相应的提示
